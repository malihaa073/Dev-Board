Question-1: What are the different ways to select an element in the DOM?

By using id,class,name,tag,query selector an element can be select in DOM.
getElementById is used when targeting a unique element ID. getElementsByClassName allows selection of multiple elements by their class, while getElementsByTagName targets elements by their tag type. querySelector is used to select the first matching element using a CSS selector, and querySelectorAll is used for selecting multiple elements with a CSS selector. getElementsByName is used to select form elements by their name attribute.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML retrieves or sets the entire HTML content inside an element, including child elements and tags. innerText Retrieves only the visible text, ignoring hidden elements. It ensures that hidden text remains invisible. textContent Retrieves the raw text content, including both hidden and visible text, without formatting.

Question-3: What is event delegation in the DOM?

When an event occurs, such as a click, event delegation allows us to handle it at a higher-level parent instead of attaching event listeners to multiple child elements. This is useful when dealing with dynamically added elements. For example, instead of adding a click event to each child element, we can attach the event listener to their parent. When a child is clicked, the event bubbles up to the parent, which then detects and handles the event accordingly.

Question-4: What is event bubbling in the DOM?

When an event occurs, such as a click, it is first triggered on the specific element where the action happens. This event then bubbles up from the target element to its parent and continues upward through the DOM hierarchy until it reaches the document object. For example, if a click event happens on a child element, it is triggered on the child first. Then, the event bubbles up to its parent, triggering the parent's event handler as well.

Question-5: How do you create, add, and remove elements using JavaScript?

By using document.createElement() we can create an element. then by using .append() .appendChild() .innerHTML() we can add element. Lastly by using .remove() and.removeChild() we can remove an element.
Here giving a example:

const CreateDiv = document.createElement("div"); //create a new div
CreateDiv.innerText = "this is a div"//add text inside it;
const container = document.getElementById("containerSection");
container.appendChild(CreateDiv);// added div inside the containerSection id
const heading = document.getElementById("head-p"); container.removeChild(heading); // Removes the heading from its parent
